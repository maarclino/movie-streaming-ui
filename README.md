# Movie Streaming UI

Frontend application that simulates a movie streaming platform interface inspired by services like Netflix.

## Description
This project is a web application built using HTML, CSS, and vanilla JavaScript. It allows users to browse a catalog of movies, view detailed information, and play trailers through an interactive interface.

The application is fully client-side and focuses on DOM manipulation, UI design, and dynamic content rendering.

## Features
- Dynamic movie catalog rendering
- Interactive movie detail views
- Trailer playback using modal windows
- Responsive layout for different screen sizes
- UI interactions handled with vanilla JavaScript

## Technologies
- HTML5
- CSS3 (Flexbox and Grid)
- JavaScript (ES6+)

## Project structure
- /css: styles and layout
- /js: application logic and data handling
- /assets: images and media resources
- index.html: main entry point

## How to run
This project requires a local server to run properly due to browser restrictions when loading local resources.

Recommended method:
- Open the project in Visual Studio Code
- Install the Live Server extension
- Right-click on index.html
- Select "Open with Live Server"

Alternative:
- Open index.html directly in a browser (some features may not work correctly due to file loading limitations)
